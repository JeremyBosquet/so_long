# so_long
42 MiniGame Project - Make 2D minigame
![image](https://i.imgur.com/Yn8QPS9.png)
# Compiling
> Run ```make```

# Running
```
./so_long <map>
```

# Maps
> Maps in folder ```maps```

> File map extension **must** be **```.ber```**

> In the projects there are already 3 maps
* 42.ber    (biggest map)
* map.ber   (medium map)
* small.ber (smallest map)

Exemple:
```
./so_long maps/map.ber
```
